# FastAPI Auth Mini App

## 🇺🇸 English

This is a small full-stack project where I built a **login authentication system** using FastAPI.

###  Description

I created a mini app that includes:

* User login authentication
* JWT (JSON Web Tokens)
* Password hashing for security
* SQLite database
* Basic frontend using HTML, CSS, and JavaScript

###  Tools & Technologies

* FastAPI
* SQLite
* JWT Authentication
* Password Hashing
* HTML, CSS, JavaScript
* UV (virtual environment management)
* Git (version control)
